デバッガーを開く方法は 3 つあります:

  • Firefox メニューの [ウェブ開発] サブメニュー (メニューバーを表示している場合や OS X では [ツール] メニュー) で [デバッガー] を選択する
  • キーボードショートカット CtrlShiftS (OS X では CommandOptionS) を押下する
  • メニューボタン ( ) を押下して [開発ツール]、[デバッガー] の順に選択する
