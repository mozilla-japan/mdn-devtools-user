インスペクタでは HTML ペインの要素の横に「イベント」という単語が表示され、イベントリスナーがバインドされています。

アイコンをクリックすると、この要素にバインドされているすべてのイベントリスナーを一覧表示するポップアップが表示されます。

各行には以下が含まれます。

  • 右向きの矢頭。クリックして行を展開し、リスナー関数のソースコードを表示します。
  • スタックを指し示すカーブした矢印。それをクリックすると、デバッガにハンドラのコードが表示されます。
  • この要素にハンドラがアタッチされたイベントの名前
  • リスナーの名前と行番号。ここをクリックして行を展開し、リスナー関数のソースコードを表示することもできます
  • イベントがバブルかどうかを示すラベル
  • イベントを定義するシステムを示すラベル。Firefox は次のように表示できます：
    • 標準の DOM イベント
    • jQuery イベント
    • React イベント
