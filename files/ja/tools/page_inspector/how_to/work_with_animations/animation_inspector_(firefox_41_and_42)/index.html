アニメーションインスペクタの UI は、Firefox 43 で改良しました。Firefox 43 以降でアニメーションインスペクタがどのようになったかを知りたい場合は、"アニメーションを扱う" のメインページをご覧ください。

Firefox 41 および 42 では、アニメーションインスペクタで以下のことができます:

  • ページ上で実行しているすべてのアニメーションの情報
  • すべてのアニメーションの再生/一時停止
  • それぞれのアニメーションの再生/一時停止/早戻し/早送り (Firefox 39 の新機能)
  • アニメーションの特定の時点にジャンプする (Firefox 39 の新機能)
  • アニメーションするノードのハイライト表示と調査 (Firefox 40 の新機能)
  • それぞれのアニメーションの再生レートを調節する (Firefox 40 の新機能)
  • アニメーションを Compositor スレッドで実行しているかを表示する (それぞれのアニメーションの隣に、稲妻のアイコンを表示します) (Firefox 41 の新機能)
